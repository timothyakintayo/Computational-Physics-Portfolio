# Computational Physics & Simulation Portfolio

This repository contains technical reports and documentation for computational physics projects completed during my Master's in Nanotechnology & Mathematical Engineering.

## 1. Monte Carlo Simulation of 2D Ising Model
**Tools:** MATLAB, Metropolis-Hastings Algorithm
**Summary:**
Designed a statistical mechanics simulation to model ferromagnetism on a square lattice (L=100) at zero external magnetic field.
* Implemented the Metropolis algorithm to simulate spin flip dynamics.
* Analyzed phase transitions, magnetization, and susceptibility near the critical temperature.
* **[View Full Project Report](./Ising_Model_Simulation/Timothy_Kinetic_Theory.pdf)**

## 3. Molecular Dynamics Simulation of Solid Argon
**Tools:** LAMMPS, Python, GNUplot
**Summary:**
Modeled interatomic interactions using the Lennard-Jones potential to compute Bulk Modulus and Specific Heat.
* Implemented NpT and NVT ensembles to simulate phase behavior under pressure.
* Analyzed thermodynamic trajectories to derive Equations of State (Birch-Murnaghan).
